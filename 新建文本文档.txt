学习体会
git
什么是git
git是世界上最先进的分布式版本控制系统
为什么要使用git
git可以提供一个仓库，在其中可以上传自己的代码文件，也可以将其中的文件下载下来，有利于多人配合完成一个项目。
2.git相对于其他系统，他可以离线提交，减少了很多的局限性。
git的配置
安装好git后，首先要在git brash界面配置自己的ID，以及邮箱
git的工作流程
1.git clone可以把远程仓库复制到本地
2.git init可以创建一个本地仓库
3.git add .可以把文件从工作目录移交到暂存区
4.git commit是编辑文件，将文件移到本地仓库（此时才可以提交文件到远程仓库）
5.vi 文件名，重新编辑文件（注意：文件经过编辑后会移交到工作目录，需要重新移交到暂存区，再移交到本地仓库，最后提交）
6.git status 查询文件的状态（注：每次操作前最好都检查一下）
git push 将文件推送到远程仓库（第一次推送时需要输入密码）
git文件状态
工作目录，暂存区，本地仓库，远程仓库
工作目录通过git add到暂存区，再通过git commit到本地仓库，最后通过git push到远程仓库，
而git pull则是把远程仓库拉到本地目录
git 分支
git branch 名字 创建分支
git branch 查看本地分支
git branch -a 查看本地和远程分支
git checkout -b prince 创立一个分支并切换到prince

C语言学习
引言：对于C语言的学习我本以为这次任务有点重复学习，毕竟原来已经通过课件学习过了，但是当我采用书本重新学习，才发现自己有许多东西还不懂。
细节1：int在不同的操作系统中的范围不同，在16位系统中可以达到几万，而在32位系统中可以达到几十亿
细节2：打印浮点型时，可以用%.2f来控制输出的小数位数。
细节3：char型可以打印较小的整数，原因是有ASCII码，其中需要注意数字和字符的区别。例如：字符4对应的ASCII码是52.'4'表示字符4，而不是数值4.
细节4：转义字符必须用单引号括起来，但是在字符串里则不需要
细节5：浮点型常量可以没有小数点或指数部分，但是不能同时省略两者。可以省略小数部分或整数部分，但是不能同时省略两者。
注意（绝不能在浮点型常量中间加空格）
细节6：用%e打印指数记数法的浮点型，而不是用%f
